# 布局容器

# 说明

Container容器使用了flex布局，默认排列方向为column，当子组件有yv-aside时，排列方向为row。

yv-container：外层容器，用来包裹下面陈列的组件；

yv-header：头部组件；

yv-aside：侧边栏组件；

yv-main：主要区域组件；

yv-footer：底部组件。
